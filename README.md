# Carpeta assets

Aqui puedes agregar imagenes mas adelante.

Ejemplo recomendado:

```text
assets/
├── despertar.png
├── dientes.png
├── desayuno.png
├── escuela.png
├── recreo.png
├── casa.png
└── dormir.png
```

La V1 no usa imagenes reales todavia. Usa bloques visuales con texto grande para que sea facil reemplazarlos despues.
