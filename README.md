# Un Dia Valiente

Juego interactivo en Streamlit para niños de 3 a 5 años.

## Que incluye

- Seleccion de personaje: nina o nino
- Campo para escribir el nombre
- Historia personalizada
- Camino completo del dia
- Decisiones con mensajes de aprendizaje
- Botones grandes
- Textos grandes
- Pantalla final con felicitacion
- Resumen para mama o papa

## Como ejecutarlo localmente

1. Instala dependencias:

```bash
pip install -r requirements.txt
```

2. Ejecuta la app:

```bash
streamlit run app.py
```

## Como subirlo a GitHub

Sube todos estos archivos a un repositorio:

```text
un_dia_valiente_streamlit_v1/
├── app.py
├── requirements.txt
├── README.md
├── .streamlit/
│   └── config.toml
└── assets/
    └── README.md
```

## Como publicarlo en Streamlit Community Cloud

1. Sube esta carpeta a GitHub.
2. Entra a Streamlit Community Cloud.
3. Crea una nueva app.
4. Selecciona tu repositorio.
5. En Main file path escribe:

```text
app.py
```

6. Deploy.
